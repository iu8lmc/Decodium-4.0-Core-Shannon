# Decodium 4 FT2 1.0.61

Release 1.0.61 raccoglie le correzioni recenti per stabilita', CAT/CI-V,
audio, UDP, logging, FT2 e interfaccia QML.

## Highlights

- FT2: sequenza TX piu' stabile dopo log/73, senza regressioni verso report gia'
  superati.
- CAT/Hamlib: CI-V Icom aggiornato automaticamente dal rig selezionato e campo
  UI solo informativo.
- IC-7300MK2: default CI-V `0xB6`.
- UDP reporting: supporto per canale terziario separato.
- Audio: refresh periferiche e livelli resi piu' coerenti tra macOS, Windows e Linux.
- UI: pannelli, menu laterale, lingua, POP, MAM queue e indicatori ripuliti.
