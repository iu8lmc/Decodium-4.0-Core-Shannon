# Decodium 4 FT2 1.0.47

Release 1.0.47 aligns the local app version, Windows installer metadata, macOS Apple Silicon packaging, and release assets.

## Highlights

- Decoder work continued on weak FT8 candidates, late decode handling, and primary decode stability.
- TCI integration was refined for TX/radio state handling and Thetis-oriented compatibility.
- Settings were cleaned up: unused options were removed or re-wired to real properties.
- Display controls now include language selection, panel visibility toggles, and dock/pop behavior for spectrum panels.
- Application icons were replaced for macOS, Windows, and Linux packaging.
- Translation coverage was completed across the bundled UI languages, with a single English translation set.
