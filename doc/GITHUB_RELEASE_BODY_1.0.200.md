# Decodium 4 FT2 1.0.200

Maintenance release with the latest local fixes.

- Hardened macOS audio shutdown paths around Qt/CoreAudio device disconnect handling.
- Improved HRD TCP compatibility diagnostics and UTF-16 reply decoding.
- Added QRZ Logbook API key test feedback in the Reporting settings panel.
- Included UI cleanup and release workflow updates for Windows, macOS Apple Silicon, and Linux Qt 6.11 x86_64 assets.

Windows, macOS, and Linux release assets are produced by GitHub Actions. Source archives are attached automatically by GitHub for tag `1.0.200`.
